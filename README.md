## Installation
## To run this project, you'll need Python and the following libraries. You can install them using pip:




## pip install spacy nltk transformers scikit-learn matplotlib graphviz
## Additionally, you'll need to download specific models and data for spaCy and NLTK:






## python -m spacy download en_core_web_sm
## python -m nltk.downloader punkt stopwords wordnet
